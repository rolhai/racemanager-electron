# racemanager-electron

## run for development

```bash
# Clone this repository
git clone https://github.com/rolhai/racemanager-electron
# Go into the repository
cd racemanager-electron
# Install dependencies
npm install
# Run the app
npm start
```

## build release

```bash
npm run package-linux
npm run package-win
npm run package-mac
```
